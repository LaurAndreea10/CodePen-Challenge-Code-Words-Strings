# 🎼 Code Words · Strings

An award-style interactive case study that explores the evolution of a creative concept through six distinct levels — from a simple interactive string experiment to a refined neon, musical, and cinematic experience.  
This project is designed as a **single-page portfolio experience** focused on **concept, process, evolution, atmosphere, interaction, and final polish**. :contentReference[oaicite:1]{index=1}

## ✨ Overview

**Code Words · Strings** plays with the layered meaning of the word **“strings”** — musical strings, programming strings, and the tension between gesture, sound, and text.  
Rather than showing isolated variations, the project presents a **clear evolution map**: where the idea started, how it transformed, and how it reached its final expressive form. :contentReference[oaicite:2]{index=2}

## 🚀 Features

- 🌍 **Bilingual interface** — supports **Romanian** and **English**

- 🧩 Clear storytelling sections:

  - **Process**
  - **Evolution**
  - **Final Result**
- 🔁 A **6-level progression** that documents the full creative journey
